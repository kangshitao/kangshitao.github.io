# 基础指令

## 创建版本库

在相关文件目录下，右键打开Git，其中Git Bash Here为命令行窗口，Git GUI Here为GUI界面，这里主要是针对命令行窗口。

```
$ mkdir learngit	# 创建learngit文件夹     $ torch xx.txt  表示创建xx.txt文件
$ cd learngit	# 进入文件夹

```
